

# Website Blog Modern (React + Vite)

Website blog sederhana dan menarik menggunakan React dan Vite. Terdiri dari halaman utama (daftar posting), detail posting, about, dan contact. Responsive dan mudah dikembangkan.

## Fitur
- Daftar posting blog terbaru
- Halaman detail posting
- Navigasi modern (Home, About, Contact)
- Responsive dan tampilan modern

## Struktur Folder
- `src/pages/` — Halaman Home, Post, About, Contact
- `src/components/` — Komponen Navbar

## Menjalankan Project
1. Install dependencies:
	```bash
	npm install
	```
2. Jalankan server pengembangan:
	```bash
	npm run dev
	```
3. Buka browser ke [http://localhost:5173](http://localhost:5173)

---
Blog ini dapat dikembangkan lebih lanjut sesuai kebutuhan Anda.
